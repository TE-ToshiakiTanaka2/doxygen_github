# doxygen_github
A short description of the project

## Installing
Install and update using pip-tools
``` bash
$ python -m venv .env
$ . .env/bin/activate.fish
(.env) $ pip install pip-tools
(.env) $ make requirements
(.env) $ pip install -e .
```

## A Simple Example
